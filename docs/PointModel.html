Class: PointModel

PointModel(point)

A PointModel is a Model centered around a specific point.

It extends Model by providing the .point property, which represents the central or representative location of the model.

Subclasses may override likelihood and sample to define the actual distributional behavior around this point.

Constructor

new PointModel(point)

Parameters:

point *

The point representing the center of the model.
