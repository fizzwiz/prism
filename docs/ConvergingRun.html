Class: ConvergingRun

ConvergingRun(start, stepper, convergenceChecker)

Base class for convergence-based iterative algorithms.

A ConvergingRun starts from an initial value and repeatedly applies a stepper function until a convergence condition is met or a maximum number of iterations is reached.

Constructor

new ConvergingRun(start, stepper, convergenceChecker)

Parameters:
Name Type Description
start *

The initial value for the iteration.
stepper function

A function that returns the next value from the current one.
convergenceChecker function

A function that compares previous and current values and returns true if convergence has been reached.
